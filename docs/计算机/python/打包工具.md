# Python 一键打包工具

为了方便大家将 Python 代码分享给他人，制作了一个可视化打包工具。无需记忆复杂的命令行，支持自动转换图片为图标。

## ✨ 功能特点
* **傻瓜式操作**：只需选择文件和图标。
* **自动转换**：支持 JPG/PNG 图片直接作为图标（自动转 .ico）。
* **独立运行**：生成的程序无需 Python 环境即可运行。

## 📥 下载地址
[点击下载 EasyPacker打包器 (Windows版)](EasyPacker.exe)

> 注意：下载后可能会被杀毒软件误报，请选择“保留”或“信任”。

## 📖 使用教程
1. 下载并打开 `EasyPacker.exe`。
2. 点击第一行的“浏览”，选择你的 `.py` 代码文件。
3. (可选) 点击第二行的“浏览”，选择一张图片作为图标。
4. 点击绿色大按钮 **“开始打包”**。
5. 等待几秒钟，打包好的 `.exe` 会自动弹出在 `dist` 文件夹中。
